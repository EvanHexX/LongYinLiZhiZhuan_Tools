# 🚀 LongYinLiZhiZhuan(Dragon's Heritage) Manual Optimization Tool v1.0.0

We are excited to announce the first official release (v1.3.14) of the **Manual Optimization Tool** for LongYinLiZhiZhuan players!
Systematically manage complex martial arts learning processes and find the most efficient path to power.

## ✨ Key Features

- **Real-time Stats & Martial Arts Management**: Monitor your character's core attributes (Strength, Agility, Intelligence, etc.) and martial arts status (Inner Power, Lightness, etc.) at a glance.
- **Goal Setting**: Set target values for specific attributes and martial arts to calculate the optimal combination of manuals.
- **Automated Recommendation List**: Generates a recommended learning sequence based on your current status and owned manuals.
- **Rarity Statistics**: Analyzes manual usage by rarity (from Basic to Divine) and tracks total progress stages.
- **Manual Database Integration**: Detailed management of manual types, sources, usage stages, and attribute growth values.

## 📦 Installation & Execution

1. Download the `lylzz_tool.zip` file from the `Assets` section below.
2. Move and unzip the downloaded file to a preferred folder and run it.
   - *Note: Ensure the `data` folder and required data files are in the same directory.*
3. If a security warning appears, click 'More Info' to proceed with the execution.

## 🛠 Improvements & Fixes (v1.3.14)
- Initial implementation of the optimization engine and UI.
- Added manual data filtering and group management features.
- Applied maximum learning limit logic by rarity.

- **실시간 속성 및 무학 관리**: 캐릭터의 현재 근력, 민첩, 지력 등 주요 속성과 내공, 경공 등 무학 상태를 한눈에 파악할 수 있습니다.
- **학습 목표 설정**: 목표로 하는 속성과 무학의 수치를 설정하고, 목표치 달성을 위한 최적의 비급 조합을 계산합니다.
- **추천 비급 목록 자동 생성**: 보유한 비급 중 현재 상태에서 가장 효율적인 비급 순서를 추천해 드립니다.

- **비급 데이터베이스 연동**: 다양한 비급의 종류, 출처, 사용 단계 및 속성 증가치를 상세하게 관리합니다.
